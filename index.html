  • Correct Answers: +2 coins per correct answer
  • Level Completion: Bonus coins based on performance
  • Perfect Levels (5/5): +5 bonus coins
  • Daily Challenge: 10-20 coins (can be doubled via ads)
  • Daily Login Bonus: 50 coins daily
